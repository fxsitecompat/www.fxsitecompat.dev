[`hyphens`](https://developer.mozilla.org/ja/docs/Web/CSS/hyphens) CSS プロパティの仕様が安定したため、接頭辞が外されました。接頭辞付きの `-moz-hyphens` プロパティは今のところ `hyphens` のエイリアスとして残されていますが、近い将来削除されます。
