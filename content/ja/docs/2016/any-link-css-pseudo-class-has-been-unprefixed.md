The [`:any-link`](https://developer.mozilla.org/ja/docs/Web/CSS/:any-link) CSS4 擬似クラスがベンダー接頭辞なしに Firefox で使用可能となりました。接頭辞付きの `:-moz-any-link` 擬似クラスへの対応は近い将来削除されます。
